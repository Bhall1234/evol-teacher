How to Use the Python Code Task Runner

Select a Task: Use the dropdown menu to select a coding task. Each task is designed to help you learn specific Python concepts.

Task Description: Once a task is selected, the task description will appear. Read it carefully to understand what the task requires.

Searching: Use any resource you want, other than chatbot's such as ChatGPT or similar, to try and fix the incorrect code.

Incorrect Code: You will be shown an example of incorrect code related to the task. Your goal is to identify and fix the errors.

Copy Code to Editor: Click the "Copy Code to Editor" button to load the incorrect code into the editor. You can then start editing the code to fix the errors.

Run Code: After fixing the code, click "Run Code" to execute it and see if it produces the correct output.

Check Code: Once you think your code is correct, click "Check Code" to verify if your solution is correct. If it is, you'll receive a follow-up challenge and a reflection question.

Can't Figure It Out: If you're stuck, you can click "Can't Figure It Out" to move on to the next task. Your attempt will be logged.

Reflection Question: If your solution is correct, you'll be presented with a reflection question. This is to help you think about what you learned. Answer it in the provided text area.

Follow-Up Challenge: You may also receive a follow-up challenge that extends the task. You can copy your current code to the follow-up editor and make the necessary modifications.

Submit: Don't forget to submit your reflections and follow-up code using the provided buttons.

Next Question: Choose the next question in the drop down menu.
